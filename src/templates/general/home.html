Secure Biomedical Analysis

This is a portal for running privacy-preserving distributed computation on genomic data. It currently supports three workflows:

  1. SF-GWAS - (Secure-Federated Genome-Wide Association Study) with Homomorphic Encryption
  2. SF-PCA - (Secure-Federated Principal Component Analysis)
  3. MPC-GWAS - (Multi-Party Computation Genome-Wide Association Study) with Secure Multi-Party Computation

1

Join

Create or join a secure study with your collaborators.

2

Configure

Configure your desired parameters and workflow.

3

Run

Run the study and view and share your results.

Go to the Workflows page to read more about each of the three workflows.
Go to the Instructions page for more information on the configuration options for running the protocol.
Go to the Studies page to get started with creating or joining a study.
