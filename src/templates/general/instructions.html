General Instructions

For each of the three workflows (SF-GWAS, SF-PCA, and MPC-GWAS), you have two options for how to set up your VM (virtual machine) and run your study:

1. The sfkit CLI.

2. Have the website set up your VM and run your study for you. This will require you to give the website limited permissions to access your Google Cloud Platform (GCP) account. (You will be walked through the specifics of this process if you create a study using this option.)

Option 1 is recommended for users who are familiar with the command line and want to have more control over their VM. Option 2 is recommended for users who are not familiar with the command line and/or want to get started quickly.
